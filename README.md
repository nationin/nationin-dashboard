# NationIn 6-Month Marketing Dashboard

Interactive dashboard tracking Nation Innovation's 6-month social media & YouTube monetization plan (May 2026 – October 2026).

🔗 **Live:** [Replace this with your GitHub Pages URL after setup]

## Features

- 📅 **Monthly calendar view** with daily tasks, milestones, and recurring weekly content
- 🎯 **Phase tracking** across 3 phases (Fix & Activate → Grow & Engage → Scale & Monetise)
- 📊 **KPI targets** for all 8 channels (YouTube, Instagram, LinkedIn, WhatsApp, Facebook, Threads, X, NI Community)
- 💰 **Revenue trajectory** chart and **monthly budget** breakdown
- ✅ **Task completion tracking** (in-session)
- 🔍 **Channel filter** to focus on a specific platform

## How it's built

Single-file React app — no build step. React 18 + Babel are loaded via CDN, the entire dashboard is one `index.html` file.

## Run locally

Just open `index.html` in any modern browser. No install needed.
